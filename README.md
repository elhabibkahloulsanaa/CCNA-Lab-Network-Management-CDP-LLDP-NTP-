# CCNA-Lab-Network-Management-CDP-LLDP-NTP-
This lab demonstrates the configuration of network management protocols on Cisco routers including CDP, LLDP, NTP and Syslog.  Technologies used: - CDP for Cisco device discovery - LLDP for vendor-neutral discovery - NTP for time synchronization - Syslog for centralized logging  All configurations were tested in Cisco Packet Tracer.
